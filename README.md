# 🔍 InfluScore Real - Busca Verdadeira

Análise de influenciadores com dados REAIS dos últimos 90 dias.

## Tecnologias
- Streamlit
- Requests
- BeautifulSoup4
- googlesearch-python
- Plotly
- lxml

## Como executar
1. Instale as dependências:
```
pip install -r requirements.txt
```
2. Execute o app:
```
streamlit run streamlit_app.py
```
